# Aura Barber - Sistema de Gestión de Barbería

Aura Barber es una plataforma web moderna para la gestión de turnos, servicios y portafolio de barbería. Diseñada con un enfoque en la experiencia del usuario (UX) y un flujo de reserva optimizado.

## 🚀 Tecnologías

* **Frontend:** React.js / Next.js
* **Estilos:** Tailwind CSS
* **Despliegue:** Vercel

## ✨ Características

* 📅 **Gestión de Turnos:** Sistema dinámico para la reserva de citas.
* 💈 **Catálogo de Servicios:** Visualización clara de cortes, tratamientos y precios.
* 📱 **Diseño Responsive:** Optimizado para dispositivos móviles y escritorio.
* ⚡ **Performance:** Carga rápida y navegación fluida.

## 🛠️ Instalación

1.  Clona el repositorio:
    ```bash
    git clone [https://github.com/tu-usuario/aura-barber.git](https://github.com/tu-usuario/aura-barber.git)
    ```
2.  Instala las dependencias:
    ```bash
    npm install
    ```
3.  Levanta el entorno de desarrollo:
    ```bash
    npm run dev
    ```
